# 🍔 Dev Burger - Landing Page de Delivery 

Uma landing page moderna, totalmente responsiva desenvolvida para uma hamburgueria artesanal com foco em conversão de pedidos via WhatsApp.

## 🚀 Funcionalidades

- **Design Moderno:** Interface limpa e profissional com foco em experiência do usuário.
- **Totalmente Responsiva:** Adaptada para disponíveis móveis, tablets, e desktop (Mobile First optimization).
- **Navegação Suave (Smooth Scroll):** Links externos que levam os usuários suavemente até a seção desejada.
- **Cardápio Dinâmico:** Grid de produtos com badges de "Novidades" e "Mais vendidos".
- **Integração com WhatsApp:** Botões de pedidos personalizados que facilitam a comunicação direta com o cliente.
- **Seção de Estatícas:** Destaque para autoridade da marca (+500 clientes, avaliações, etc).
