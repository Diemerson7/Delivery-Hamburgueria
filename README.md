# 🍔 DevBurger - Landing Page de Delivery

Uma landing page moderna, atraente e totalmente responsiva desenvolvida para uma hamburgueria artesanal com foco em conversão de pedidos via WhatsApp.

## 🚀 Funcionalidades

- **Design Moderno:** Interface limpa e profissional com foco na experiência do usuário.
- **Totalmente Responsiva:** Adaptada para dispositivos móveis, tablets e desktop (Mobile First optimization).
- **Navegação Suave (Smooth Scroll):** Links internos que levam o usuário suavemente até a seção desejada.
- **Cardápio Dinâmico:** Grid de produtos com badges de "Novidade" e "Mais Vendidos".
- **Integração com WhatsApp:** Botões de pedido personalizados que facilitam a comunicação direta com o cliente.
- **Seção de Estatísticas:** Destaque para autoridade da marca (+500 clientes, avaliações, etc).

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica para melhor SEO.
- **CSS3:** Estilização avançada utilizando:
  - **Flexbox & CSS Grid:** Para layouts complexos e responsividade.
  - **Variáveis CSS:** Para facilitar a manutenção de cores e bordas.
  - **Google Fonts:** Utilização da fonte *Poppins* para uma tipografia moderna.
- **Responsividade:** Media Queries customizadas para garantir centralização e usabilidade em telas pequenas.

## 📱 Visualização Mobile

A página foi otimizada para:
- Menu compactado e centralizado.
- Botões em largura total para facilitar o toque (Tap targets).
- Elementos empilhados verticalmente em telas menores para melhor leitura.

## 📦 Como usar o projeto

1. Clone ou baixe este repositório.
2. Certifique-se de que os arquivos `index.html` e `styles.css` estejam na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador.
